# Recording Guide — Slabstack Onboarding Video Series

## Pre-Recording Checklist

- [ ] Scripts reviewed and printed or open on a second screen
- [ ] Loom installed and tested (do a quick test recording)
- [ ] Browser open to app.slabstack.com and logged in
- [ ] Do Not Disturb enabled on Mac (notifications off)
- [ ] Clean desktop background (no distracting files or icons)

---

## Recording Workflow

Repeat these steps for each video in the series.

**Step 1: Generate voiceover**
Paste the script into ElevenLabs, select your voice, and download the MP3.

**Step 2: Record screen**
Open Loom and record your screen silently, following along with the script at the same pace as the audio.

**Step 3: Create a new Descript project**
Open Descript and start a new project for this video.

**Step 4: Import files**
Drop the Loom video file and the ElevenLabs MP3 into the Descript project.

**Step 5: Sync and trim**
Align the audio to the video, then trim any dead space at the beginning or end.

**Step 6: Export**
Export the finished video as an MP4.

**Step 7: Upload to Loom**
Upload the exported MP4 to Loom for sharing with customers.

---

## Tips for Clean Recordings

- **Go slow on clicks.** The AI voice moves faster than you expect — your screen actions need to keep up.
- **Zoom into key UI elements** before clicking so viewers can clearly see what you're selecting.
- **Stick to one workflow per video.** Each video should cover exactly one task from start to finish.
- **Do a 30-second test first.** Record a short clip, import it into Descript, and check that audio sync looks right before doing the full recording.
