## Script

---

`[SHOW: Slabstack dashboard]`

Welcome back. Before you create your first project or quote in Slabstack, there's one thing worth doing first: configuration.

Configuration is where you set your defaults — the buyers, products, terms, and line items that automatically populate every project and quote you create. Spend five minutes here now, and you'll save that time on every single record going forward.

Let's walk through it.

---

### Part 1: Project Configuration

`[CLICK: Settings in the left-hand navigation]`
`[SHOW: Settings menu or page]`

Start in **Settings**. Project configuration is organized by location — so if your operation runs out of multiple plants or regions, you can set different defaults for each one.

`[CLICK: the relevant Location]`
`[SHOW: Location settings page]`
`[CLICK: Project Configuration tab or section]`
`[SHOW: Project Configuration settings]`

Inside your location, find **Project Configuration**.

This is where you set three things that will default onto every new project automatically.

`[ZOOM: Buyers / Customers field]`

First, **Buyers** — these are the customers most commonly associated with projects at this location. Add them here and they'll be pre-populated when you open a new project, so you're not starting from scratch every time.

`[ZOOM: Other Companies field]`

Next, **Other Companies** — any additional organizations tied to your projects, like general contractors or owners. Set your common ones here.

`[ZOOM: Products (Mixes) field]`

And finally, **Products** — your concrete mixes. Add the mixes your location uses most often. They'll show up ready to select when you're building out a project.

`[CLICK: Save]`

When you're done, save your settings.

---

### Part 2: Quote Configuration

`[SHOW: Settings — same location selected]`
`[CLICK: Quote Configuration tab or section]`
`[SHOW: Quote Configuration settings]`

Now click into **Quote Configuration** — same location, just one section over.

This is where you set what defaults onto every quote you generate.

`[ZOOM: Terms and Conditions field]`

First, **Terms and Conditions**. Add your standard T&Cs here — the language you include on every quote. Once it's saved, it will populate automatically so you're never hunting for it or pasting it in manually.

`[ZOOM: Additional Items field]`

Next, **Additional Items** — things like fuel surcharges, delivery fees, or any standard line items you add to most quotes. Set them here and they'll be waiting for you every time.

`[CLICK: Save]`

Save when you're finished.

---

`[SHOW: Slabstack dashboard or a new blank project/quote screen]`

That's configuration. It takes just a few minutes to set up, but it changes how fast you can work from here on out — every project and quote you create will already have your defaults in place, ready to go.

In the next video, we'll cover Activities — how to log calls, track notes, and keep a clear record of everything happening on your accounts.

See you there.
